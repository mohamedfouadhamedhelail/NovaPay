# 💳 NovaPay

NovaPay is a simple **C# Console Application** built to practice the fundamentals of **Object-Oriented Programming (OOP)**.

The current version allows users to create a bank account and view the account information through a simple console interface.

---

## 🚀 Features

- Create a new bank account
- Store account information using an `Account` object
- Display account details
- Simple console-based menu
- Object-Oriented project structure

---

## 🛠️ Technologies Used

- C#
- .NET
- Console Application
- Object-Oriented Programming (OOP)
